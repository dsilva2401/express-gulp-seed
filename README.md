# express-gulp-seed

An easy way to start with express mvc using gulp

```bash
mkdir my-app
cd my-app
git clone https://github.com/dsilva2401/express-gulp-seed.git .
npm install
gulp develop
```

#### Create module
```bash
gulp create-module --name login
gulp develop
```

#### Remove module
```bash
gulp remove-module --name login
gulp develop
```
